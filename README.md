# Oasis Infobyte Cybersecurity Internship

## Overview

This repository contains all tasks completed during my 1-month Cybersecurity Internship at Oasis Infobyte. The internship focused on practical cybersecurity concepts including network scanning, firewall configuration, web application security testing, threat analysis, and social engineering awareness.

## Objectives

* Gain hands-on experience with cybersecurity tools.
* Learn vulnerability assessment techniques.
* Understand network security fundamentals.
* Develop practical skills in ethical hacking and security analysis.

## Tasks Completed

### Task 1: Nmap Network Scanning

**Objective:** Perform network reconnaissance and identify active hosts, open ports, and running services using Nmap.

**Tools Used:**

* Nmap

**Key Learnings:**

* Host discovery
* Port scanning
* Service enumeration

---

### Task 2: Basic Firewall Configuration

**Objective:** Configure and manage firewall rules to control incoming and outgoing network traffic.

**Tools Used:**

* UFW (Uncomplicated Firewall)

**Key Learnings:**

* Firewall configuration
* Allowing and blocking ports
* Network access control

---

### Task 3: SQL Injection Testing

**Objective:** Understand SQL Injection vulnerabilities and learn methods to identify and prevent them.

**Tools Used:**

* DVWA (Damn Vulnerable Web Application)

**Key Learnings:**

* SQL Injection concepts
* Input validation
* Secure coding practices

---

### Task 4: Network Security Threat Analysis

**Objective:** Research common network security threats and mitigation strategies.

**Topics Covered:**

* Malware
* Phishing
* Denial of Service (DoS)
* Man-in-the-Middle (MITM) Attacks

---

### Task 5: Social Engineering Research Report

**Objective:** Study various social engineering techniques and methods used by attackers.

**Topics Covered:**

* Phishing
* Spear Phishing
* Pretexting
* Baiting
* Tailgating

---
